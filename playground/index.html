◗ PecanX

A correctness-first language for full-stack web apps. Pure logic compiles to one shared Kernel (JS + WebAssembly); interface code compiles to JavaScript. This whole page runs the real pcx compiler in your browser — nothing is sent to a server.

It is designed to make four bug families unrepresentable:

  • null / undefined → Option<a>
  • unhandled errors → Result<e, a> (no exceptions)
  • impossible UI states → exhaustive sum-type match
  • client/server drift → one shared Kernel validates both sides
